### django基础
1、掌握 Django 项目的创建、配置及启动
2、掌握 Django 调度器，可根据需求，同时结合 path() 或 re_path() 函数来定义 URL 调度器
3、掌握 Django 应用的创建
4、掌握 Django 模板的创建
5、能够流畅的表达出 Django 调度器的工作流程

#### 1.创建虚拟环境
```bash
  python -m venv 名称
  激活在scripts里
```
#### 2.创建django项目（推荐用pycharm来创建）
```bash
  python -m django startproject 名称
  python manage.py runsever(启动项目)
  http://127.0.0.1:8000（浏览器中可以查看）
```
- **配置文件**
```bash
练习/
├── 练习
│ ├── asgi.py       （ASG I 兼容的 Web 服务器上的入口）
│ ├── __init__.py   （空文件代表的是告诉解释器这是一个python包）
│ ├── settings.py   （django项目的配置文件)
│ ├── urls.py       （url路由配置文件）
│ └── wsgi.py       （WSGI 兼容的 Web 服务器上的入口）
└── manage.py       （管理 Django 项目的命令行工具）
```
```bash
setting中的更改配置
```

#### 3.URL调度器
- 这个模块是纯粹的 Python 代码，包含 URL 模式（简单的正则表达式）到 Python 函数（视图）的简单映射。
- URL 调度器也被称为 URL 路由或者 URLConf 或者 URL 路由分发器。
```bash
# 配置路由
path('hello/',views.hello),
# 设计视图
from django.http import HttpResponse
def hello(request):
return HttpResponse('Hello,Django')
```
