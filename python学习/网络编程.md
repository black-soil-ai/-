### TCP编程
- socket(套接字)（把复杂的TCP和IP协议封装好）：用于计算机在网络通信中的一个接口，不同计算机进行数据交换和通信
#### 内置模块socket
- 1.创建服务端的socket对象`socket.socket(family,type)`
- 2.绑定服务端的IP和端口号 `bind((ip,port))参数必须是一个元组（ip，port）`
- 3.设置监听模式，允许多少个客户端连接`server.listen(2)`
- 4.等待客户端连接`conn, client = server.accept()`
- 5.接收客户端发来的请求`data = conn.recv(1024)`
- 6.给客户端返回/发送响应数据`conn.send("洞幺（服务端）收到！洞幺（服务端）收到！".encode("utf-8"))`
- 7.关闭/断开连接`conn.close()`
- 8.关闭服务器`server.close()`
- （socket->bind->listen->accept->recv->send->close）
- family:网络地址类型（ipv4/ipv6） AF_INET:ipv4/AF_INET6:ipv6
  type:socket传输类型（tcp/udp）  SOCK_STREAM:TCP/SOCK_DGRAM:UDP
```bash
accept() 有2个返回值
    返回值1：与客户端连接的socket对象
    返回值2：客户端地址
```
