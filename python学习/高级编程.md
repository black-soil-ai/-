## 一. 面向对象（类和对象）
1. 面向对象思想（不同对象中封装这不同的许多方法）（三大特征：封装，继承和多态）
```bash
  # 创建类
  Class human():
      age='gender'
  # 创建对象（实例化）
  p1=human()
```
2. 类、对象的属性和方法
```bash
  Class human():
    name="人类",
    def eat(self):
        print('我将打造一个全盛时期的我自己')

  p1=human()
  # 添加对象的属性
  p1.age='gender'
  print(p1.__dict__) # 对象的'属性字典'（不包括继承来的）
  # 调用方法
  p1.eat()
```
